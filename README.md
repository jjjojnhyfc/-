妈妈做菜太好吃每天不知道吃什么怎么办！那就用家庭点菜系统找找每天想吃什么吧！

后端运用的技术栈：Springboot+maven+mybatis+MySQL

前端运用的技术栈：Veu3+ts+naiveui
